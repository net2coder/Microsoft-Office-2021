### Step 1: Install Office
1. Open this folder in PowerShell or Command Prompt
2. Right-click in the empty space and select "Open PowerShell here"
3. Run this command:
   ```
   .\setup.exe /configure .\configuration-Office2021Enterprise.xml
   ```
4. Wait for the installation to complete

### Step 2: Activate Office
1. Open PowerShell as Administrator
2. Follow the commands in `activation-steps.txt`
3. Done! Your Office 2021 is now activated
